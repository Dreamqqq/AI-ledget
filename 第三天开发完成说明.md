## 技术要点

### 1. BCrypt 密码加密
```java
private final BCryptPasswordEncoder passwordEncoder = new BCryptPasswordEncoder();
String encodedPassword = passwordEncoder.encode(plainPassword);
```

BCrypt 特点：
- 自动加盐
- 每次加密结果不同
- 单向加密，无法解密
- 验证时使用 `passwordEncoder.matches(plainPassword, encodedPassword)`

### 2. 手机号验证正则
```
^1[3-9]\\d{9}$
```
- `^1`: 以 1 开头
- `[3-9]`: 第二位是 3-9
- `\\d{9}`: 后面 9 位数字
- `$`: 结束

### 3. 参数验证注解
- `@NotBlank`: 不能为空且不能是纯空白字符
- `@NotNull`: 不能为 null
- `@Pattern`: 正则表达式验证
- `@Valid`: 触发验证

### 4. 事务管理
```java
@Transactional
public RegisterResponse register(RegisterRequest request) {
    // 数据库操作会在事务中执行
}
```
